Register common custom post typess and meta fields.

== Description ==

This plugin is meant to used by some of themes by Themeglory. This plugin registers required custom post types and meta fields to make theme able to work.

Note: Some of the features are not available on free version of the theme.

This plugin creates sub menu 'Themeglory Tool' on wordpress admin panel under settings. Enable 'Custom post types' according to your requirement.
